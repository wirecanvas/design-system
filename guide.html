    AI에게 회사의
    디자인 규칙을
    딱 두 파일로
    알려주세요.

    영업팀, 운영팀에서 Claude로 사내 서비스를 만드실 때 회사 디자인 톤앤매너를 자동으로 맞추는 방법입니다. 한 번만 익히면 평생 쓸 수 있어요.

    소요 시간
    약 5분
    필요한 것
    Claude.ai 계정
    결과물
    통일된 톤의 사내 서비스

    이런 분께 추천해요

    • Claude로 업무 자동화 서비스를 만들고 있는데, 디자인이 매번 다르게 나와서 고민이신 분
    • 내가 만든 서비스가 회사의 다른 서비스랑 자연스럽게 어울렸으면 하는 분
    • "디자인은 잘 모르지만 결과물은 깔끔했으면 좋겠다"고 생각하시는 분

    핵심 한 줄. AI에게 두 개의 파일(tokens.css, typography.css)을 첨부하고 "이 파일의 규칙만 써줘"라고 말하면 끝입니다.

    참고 — 이 사이트도 같은 디자인 시스템으로 만들었어요

    이 페이지의 모든 색·폰트·간격·그림자는 회사의 tokens.css와 typography.css에서 가져온 값입니다.

    STEP 01 PREPARE

    받으신 두 개의 파일을
    확인하세요.

    디자인팀에서 받으신 파일을 한 곳에 모아두세요. 두 파일이 모두 필요합니다.

    CSS
    tokens.css
    색상, 간격, 그림자를 담은 디자인 토큰 파일
    색상 182개 간격 / 크기 84개 그림자 28개
    CSS
    typography.css
    글씨 스타일을 클래스로 정리한 파일
    제목 스타일 (h1~h6) 본문 (body-1, body-2) 총 33개 클래스

    열어보지 않으셔도 돼요. 직접 코드를 읽거나 수정할 필요 없습니다. AI에게 첨부만 하면 알아서 사용합니다.

    새 대화를 시작하세요.

    03

    화면 가운데 또는 아래에 메시지 입력창이 나타납니다.
    STEP 03 ATTACH FILES

    두 파일을 함께
    첨부하세요.

    입력창 안에 두 개의 CSS 파일을 모두 올려주세요. 두 파일 중 하나라도 빠지면 디자인이 완전하게 적용되지 않습니다.

    01
    첨부 아이콘 클릭
    입력창 왼쪽 또는 아래의 📎 클립 아이콘(또는 +)을 클릭합니다.
    02
    tokens.css 선택
    파일 선택 창에서 tokens.css를 선택해 업로드합니다.
    03
    typography.css 선택
    같은 방식으로 typography.css도 올립니다. 한 번에 두 파일을 드래그앤드롭으로 올려도 됩니다.
    04

    입력창 위에 두 파일이 작은 칩 형태로 보이는지 확인합니다.

    같은 대화에서는 한 번만 첨부하면 됩니다. 새 대화를 열 때는 다시 첨부해야 해요.

    기본 템플릿

    아래 텍스트를 복사해서 Claude 입력창에 붙여넣으세요. 대괄호 안의 내용만 바꾸면 됩니다.

    첨부한 tokens.css와 typography.css의 디자인 토큰만 사용해서 [원하는 서비스 설명]을 만들어주세요. 규칙: - 색상은 반드시 var(--color-...) CSS 변수로 사용 - 폰트는 반드시 .text-h1, .text-body-1 같은 클래스로 적용 - 임의의 색상값(#5578F0 같은 hex)이나 폰트 크기는 사용 금지 - 결과는 단일 HTML 파일로, <style> 태그 안에 tokens.css와 typography.css 내용을 포함해서 만들어주세요

    STEP 07 FAQ

    자주 묻는 질문

    사용하면서 자주 마주치는 질문들과 답변이에요.

    CSS 파일이 뭔지 모르는데 써도 되나요?

    네, 괜찮습니다. 직접 열어보거나 수정할 필요 없이, AI에 첨부만 하면 됩니다. AI가 알아서 읽고 사용해요.

    두 파일을 매번 다시 첨부해야 하나요?

    같은 대화 안에서는 한 번만 첨부하면 됩니다. 다음 메시지부터는 자동으로 참조해요. 새 대화를 시작하면 다시 첨부해야 합니다.

    원하는 색이 파일에 없으면 어떻게 하나요?

    디자인팀에 문의해주세요. 새로운 색상이 정말 필요한 경우 디자인 시스템에 추가한 뒤 파일을 업데이트해드립니다. 임의로 색을 추가해서 쓰시면 일관성이 깨져요.

    ChatGPT나 다른 AI에서도 쓸 수 있나요?

    네, 같은 방식으로 사용 가능합니다. 파일 첨부와 프롬프트만 동일하게 적용하시면 됩니다. 다만 결과물 품질은 Claude가 가장 안정적이었어요.

    결과가 마음에 안 들면?

    Claude에게 자연어로 수정 요청을 계속할 수 있습니다. 단, 수정 요청에도 "tokens.css와 typography.css 변수만 사용해주세요"라는 문구를 유지해주세요. 안 그러면 AI가 마음대로 다시 색을 바꿔버려요.

    파일이 업데이트되면 어떻게 하나요?

    디자인 시스템이 변경되면 새 버전의 파일을 받게 됩니다. 항상 가장 최신 파일을 사용해주세요. 옛날 파일을 쓰면 새 색상이 빠질 수 있어요.

    여러 화면이 있는 서비스도 가능한가요?

    네, 가능합니다. 한 번의 요청에 여러 화면을 만들어달라고 할 수도 있고, 같은 대화 안에서 하나씩 추가로 요청할 수도 있어요. 같은 대화 안에서 만들면 자연스럽게 같은 톤이 유지됩니다.

    완성된 결과물을 실제로 어떻게 써요?

    Artifact 패널 우측 상단의 다운로드/복사 버튼을 통해 HTML 파일을 받을 수 있어요. 이걸 그대로 브라우저에서 열거나, 내부 서버에 올려서 사용하시면 됩니다.
